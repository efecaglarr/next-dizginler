This is a rental car website [Dizginler Rent A Car](https://nextjs.org/) 

## Dizginler Car Rent Site

To run this project

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`Manrope`](https://fonts.google.com/specimen/Manrope) 

## Specs

With this project user can view the car listings and contact the renter via mail or whatsapp.

- `Booking` and `ReservationDetails` are used to send e-mails to the renter about renting information.
- With `CarCard` and `CarDetails` user can see detailed information about the cars.
- `CustomButton` can use for more than one purpose.


## Possible Features

- Admin dashboard to manage cars and reservations.
- Database operations can be implemented.
